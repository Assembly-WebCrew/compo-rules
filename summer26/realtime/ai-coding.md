# AI Coding (Vibe Demo)

Refer to the [General rules](../general.md) with the following additions:

Other competitions do not allow AI-generated content. This one does - AI tools may be used to write up to 100% of the code. Use as much or as little AI as you want. Other assets (music, graphics) can be human-made or AI-generated as you choose. The result must be an executable real-time production, not a single-shot prompt output.

- Fan art using copyrighted characters or works is not allowed.
- AI use in code is welcome - anywhere from a little assistance to fully AI-written.
- The entry must be an **executable real-time production**. Any platform (Desktop, oldskool, fantasy console, etc.) is welcome.
- Entries may not exceed 5 minutes in length.
- Submit the executable. A 1080p video capture is also required - the video is what gets shown on screen, but the executable must be there too so we can verify it runs.
- The size of the entry must be reasonable and can be delivered via USB stick or through Partyman.
- Submit a `readme.txt` describing the AI tools used.
- A **screenshot** image must be submitted with the entry.
- The production must show in a similar manner every time it is executed.
- You must hide the **mouse cursor**.
- Entries will be recorded at 1080p60 (or possibly 1080p50) resolution.

Remote entries are welcome.
